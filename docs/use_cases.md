# IRAI Use Cases

## Home Automation (LAN-Only)

Control smart home devices over the local network without any cloud dependency. Supports lights, switches, thermostats, and other LAN-accessible devices.

## Car Assistant (No Cellular)

Run IRAI on in-vehicle hardware for navigation hints (cached maps), music control, and hands-free operation — no cellular connection needed.

## Privacy-Critical Environments

All processing stays on-device. No data leaves the local machine. Suitable for sensitive environments where cloud transmission is prohibited.

## Emergency Backup Assistant

When internet connectivity is unavailable (outages, disasters, remote locations), IRAI continues to function with cached knowledge and local processing.
